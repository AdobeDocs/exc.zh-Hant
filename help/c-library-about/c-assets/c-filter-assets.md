# 篩選資產{#filter-assets}

您可以在 Adobe Experience Cloud 資料庫中篩選資產。

若要在 Experience Cloud 資料庫中篩選資產:

1. 按一下漏斗圖示，即可篩選資產。

   ![](assets/library_filter_assets.png)

1. 選擇要用以篩選資產的選項。
