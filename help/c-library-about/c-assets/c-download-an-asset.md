# 下載資產{#download-an-asset}

您可以在Adobe Experience Cloud Library中下載資產。

1. 按一下資產。
1. 按一下資產旁的 **[!UICONTROL 「更多選項]** 」功能表(橢圓形)。

   ![](assets/library_asset_options.png)

1. 按一下 **[!UICONTROL 「下載]** 」以下載資產。
