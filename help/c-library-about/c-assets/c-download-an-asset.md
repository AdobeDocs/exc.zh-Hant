# 下載資產{#download-an-asset}

您可以在 Adobe Experience Cloud 資料庫中下載資產。

1. 按一下資產。
1. 按一下資產旁的&#x200B;**[!UICONTROL 更多選項]**&#x200B;選單 (刪節號)。

   ![](assets/library_asset_options.png)

1. 按一下&#x200B;**[!UICONTROL 「下載」]** 即可下載資產。
