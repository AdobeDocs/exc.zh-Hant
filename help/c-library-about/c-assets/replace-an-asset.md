# 取代資產{#replace-an-asset}

您可以在Adobe Experience Cloud Library中取代資產。

若要取代Experience Cloud Library中的資產：

1. 按一下資產。
1. 按一下資產旁的 **[!UICONTROL 「更多選項]** 」功能表(橢圓形)。

   ![](assets/library_asset_options.png)

1. 按一下 **[!UICONTROL 「取代]**」。
1. 選取新資產以取代舊資產。
