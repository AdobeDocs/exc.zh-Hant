# 搜尋資產{#search-for-assets}

在Adobe Experience Cloud Library中搜尋資產，以便在組織或特定資料夾中尋找資產。

使用搜尋列，在Experience Cloud Library中搜尋資產。搜尋會查看檔案名稱。
