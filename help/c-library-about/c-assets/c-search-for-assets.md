# 搜尋資產{#search-for-assets}

在 Adobe Experience Cloud 資料庫中搜尋資產，以便在整個組織或特定資料夾中找到資產。

使用搜尋列在 Experience Cloud 資料庫中搜尋資產。搜尋功能會查看檔案名稱。
