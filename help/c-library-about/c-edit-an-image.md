# 編輯影像{#edit-an-image}

使用 Adobe Experience Cloud 資料庫內建的基本影像編輯器，直接從資料庫即時編輯影像。

若要從 Experience Cloud 資料庫編輯影像:

1. 按一下要編輯的影像。
1. 按一下「編輯影像」。儲存影像時，影像會儲存為新版本。

   影像編輯器開啓後，您即可開始編輯影像。
