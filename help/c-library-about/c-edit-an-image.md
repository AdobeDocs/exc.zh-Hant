# 編輯影像{#edit-an-image}

使用Adobe Experience Cloud Library內建的基本影像編輯器，直接從資料庫編輯影像。

若要從Experience Cloud Library編輯影像：

1. 按一下影像以進行編輯。
1. 按一下編輯影像。儲存影像時，它會儲存為新版本。

   影像編輯器隨即開啓，您可以編輯影像。

   ![](assets/library_image_editor.png)
