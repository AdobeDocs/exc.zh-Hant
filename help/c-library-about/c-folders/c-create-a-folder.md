# 建立檔案夾{#create-a-folder}

您可以在Adobe Experience Cloud Library中建立資料夾來組織資產。

若要在Experience Cloud Library中建立資料夾：

1. 選擇 **[!UICONTROL 「新增]** &gt; **[!UICONTROL 資料夾]**」。

   ![](assets/library_new_folder_upload.png)

1. 輸入資料夾名稱，然後選取 **[!UICONTROL 「建立]**」。
