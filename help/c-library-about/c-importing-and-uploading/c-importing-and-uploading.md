# 匯入和上傳檔案{#import-and-upload-files}

從Dropbox或Creative Cloud匯入檔案，並從devicice上傳檔案，以便在Adobe Experience Cloud Library的一個位置管理並儲存檔案。

您必須先設定檔案夾，才能匯入或上傳檔案。
