# 匯入和上傳檔案{#import-and-upload-files}

從 Dropbox 或 Creative Cloud 匯入檔案，並從您的裝置上傳檔案，以便在 Adobe Experience Cloud 資料庫的同一位置管理和儲存這些檔案。

您必須先設定資料夾，才能匯入或上傳檔案。
